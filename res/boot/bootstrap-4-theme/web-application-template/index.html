A static web page is a page that is always displayed in front of the user in an unchanged form. The web server sends the page at the request of the web browser without any changes.

In contrast, the server makes changes to the dynamic web page before sending it to the browser. Due to the fact that the page is changing, it is called dynamic.

A web server is software that provides web pages in response to requests from web browsers.

When the web server receives a request to open a static page, then, after getting the request, the server finds the page and sends it to the browser.
